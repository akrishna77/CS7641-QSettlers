Java Settlers of Catan - Account Creator

Sorry, you need Java to create an account.
Either add the Java plugin to your browser,
or download the JSettlers JAR file and run this command:
java -cp JSettlers.jar soc.client.SOCAccountClient www.example.com 8880
This applet connects to www.example.com port 8880
Back

This is where you can create an account on the Java SoC system. Just fill in the boxes and press the "Create Account" button. I suggest including your email address so that if you forget your password, it can be used to confirm your identity.

Note: Passwords must be 10 characters or less.
